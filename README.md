# Explainable Anomaly Intelligence using CADEN - Contextual Anomaly Detection and Explanation Narrative

CADEN is a decision intelligence framework that transforms raw anomaly signals into
clear, actionable narratives for business stakeholders.

## Key Capabilities
- Governed multivariate anomaly detection
- Contextual signal enrichment
- Bayesian causal hypothesis ranking
- Executive narrative generation using LLMs
- Decision confidence KPIs (NCI, DER, CDS)

## Project Structure
- core/        → CADEN pipeline stages
- utils/       → configuration & helpers
- data/        → synthetic datasets (optional)
- main.py      → pipeline entry point
